#  Student Performance Analysis using Machine Learning

## About the Project

This project analyzes student performance using machine learning techniques. The objective is to understand the factors that influence academic outcomes and identify patterns among students based on study habits, attendance, motivation, resources, and exam performance.

The project combines both supervised and unsupervised learning approaches to predict final grades and segment students into meaningful groups.

---

## Dataset Features

The dataset contains information such as:

- Study Hours
- Attendance
- Resources
- Extracurricular Activities
- Motivation
- Internet Access
- Gender
- Age
- Learning Style
- Online Courses
- Discussions
- Assignment Completion
- Exam Score
- Educational Technology Usage
- Stress Level
- Final Grade

## Key Findings

- Exam Score was the most influential factor in predicting final grades.
- Random Forest achieved the highest classification accuracy.
- Student groups were successfully identified using clustering techniques.
- Different clustering algorithms produced varying segmentation patterns.
